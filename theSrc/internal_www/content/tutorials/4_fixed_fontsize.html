Fixed Sized Fonts

Font size is fixed to the specified pixel value in the config, as the widget resizes the images and other shapes will grow/shrink, but the font size remains the same

Other font units, such as 'em' should not be used as they are not explicitly supported. They might work, but dont do it.

{
  "font-size": "10px",
  "table": {
    "rows": [
      [ "label:A", "label:A", "label:A", "label:A", "label:A" ],
      [ "label:A" ],
      [ "label:A" ],
      [ "label:A" ],
      [ "label:A" ]
    ]
  }
}
    
{
  "table": {
    "rows": [
      [
        { "type": "graphic", "value": { "variableImage": "circle:lightblue", "text-overlay": { "text": "A", "font-size": "10" } } },
        { "type": "graphic", "value": { "variableImage": "circle:lightblue", "text-overlay": { "text": "A", "font-size": "10px" } } }
      ]
    ]
  }
}
