Table with Flexible Dimensions Tutorial

In addition to explicitly specifying row and column sizes by providing a numeric value, three additional string values can be provided to produce cells that change size depending on their content:
  • proportion:NUMBER: For example "proportion:0.25" will allocate 25% of the total width/height to the cell
  • flexible:label: Allocate the minimum space required to display the text provided (accounting for font-size, font-family, font-weight, etc.)
  • flexible:graphic: Allocate the maximum space while preserving aspect ratio of specified images
Currently the pictograph library does not support using flexible columns and rows in the same table, you must pick one.
